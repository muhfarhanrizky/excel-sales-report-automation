# excel-sales-report-automation
Automated Excel sales report that processes POS sales data to calculate key metrics such as total sales, total orders, quantity sold, AOV, AUT, and product performance.

This project demonstrates how Excel can be used to automate sales reporting using data exported from a Point of Sale (POS) system.

The Excel file processes raw sales data from the POS and automatically generates key sales metrics, including:

- Total Sales
- Total Orders
- Total Quantity Sold
- Sales by Cup Size
- Topping Sales
- Sales of Targeted Products
- Average Order Value (AOV)
- Average Units per Transaction (AUT)

By simply pasting the sales data exported from the POS system into the Excel sheet, all calculations and reports are automatically generated without manual computation.

This project was inspired by my experience working in a beverage retail environment, where daily sales reporting often requires manual calculations.
